# Roadmap

ProofPack is focused on becoming the reference OSS evidence format and verifier for AI agent runs.

## Now

- Finish the Evidence Workbench UI revamp.
- Harden shared archive and redaction paths.
- Keep CLI, API, and web verification results identical.
- Improve public docs and launch material.
- Add deterministic visual verification to prevent frontend regressions.

## Next

- Publish package artifacts from the monorepo.
- Add richer runtime adapters for agent frameworks.
- Add more fixture packs for failure modes, trust stores, multisig, timestamp anchors, and public disclosures.
- Add browser-side local verification for smaller packs.
- Add Hyperframes explainer video assets after the UI stabilizes.

## Later

- Evaluate Rust/WASM verifier only after TypeScript verifier benchmarks show a real need.
- Explore transparency-log or TSA integrations for timestamp anchors.
- Explore hosted coordination workflows without weakening offline verification.
