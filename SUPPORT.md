# Support

ProofPack is currently a pre-1.0 open-source project.

## Where To Ask

- Bug reports: use the GitHub bug report template.
- Feature requests: use the GitHub feature request template.
- Security issues: follow [SECURITY.md](./SECURITY.md), do not open a public issue.
- Design or product discussion: open a GitHub discussion or issue with the workflow you are trying to support.

## What To Include

- ProofPack version or commit SHA.
- Node and pnpm versions.
- Whether the issue appears in CLI, API, web, or core.
- A minimal pack fixture if it can be shared.
- The exact command or browser flow that failed.

## Response Expectations

This is not a staffed support desk. High-quality reproduction details get priority because they help maintainers verify and fix issues quickly.
