# Code of Conduct

ProofPack uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Standard

Be direct, technical, and respectful. Assume contributors are here to improve verifiable AI systems. Harassment, personal attacks, and bad-faith disruption are not acceptable.

## Scope

This applies to issues, pull requests, discussions, docs, examples, and any project-maintained communication channel.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior harms the project or its contributors.

Report problems through the private contact path listed in [SECURITY.md](./SECURITY.md) when the issue is sensitive, or open a maintainer-visible issue when it is not.
